Improved26 Apr 2026 · 01:44

Long matches count for more than blowouts

A 26-game battle (e.g. 7-6 7-6) tells the system more about both players than a 12-game blowout (6-0 6-0), so it now updates ratings accordingly.

More detail

Average doubles match is around 18 games. Matches noticeably longer or shorter than that scale the rating change up or down (within sensible bounds). Walkovers count for less.

New26 Apr 2026 · 00:57

First version of the doubles ranking system

Every player who's played a VLTC doubles tournament now has a skill rating that updates automatically after every match they play.

More detail

The rating uses an algorithm called OpenSkill (a modern descendant of TrueSkill). It tracks both how strong it thinks you are (μ) and how confident it is in that estimate (σ). The 'μ - 3σ' number on the leaderboard is a conservative ranking — it under-estimates new players until they've played enough matches.
